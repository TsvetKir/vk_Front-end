# vk_Front-end

Профильное задние для стажировки в команде фронтенд разработчиков

## Запуск 

Клонирование репозитория

```sh
 git clone https://github.com/TsvetKir/vk_Front-end.git
```

Установка зависимостей 

```sh
  npm install
```

Запуск проекта

```sh
 npm run start
```
